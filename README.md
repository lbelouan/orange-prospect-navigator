# Orange Prospect Navigator

**Orange Prospect Navigator** is a B2B relationship intelligence prototype designed to help sales teams better understand complex target accounts and navigate complex buying environments.

Developed with **Orange Business managers** during the **EDHEC Open Innovation Challenge**, the project explored how **AI can improve B2B relationships** by transforming account mapping into concrete commercial execution.  
The project received both the **Game Changer Award** and the **People’s Choice Award**.

## Overview

The application helps users map a target account’s stakeholder ecosystem, identify key decision-makers and influencers, and generate relevant introduction paths and actionable next steps for prospecting and account strategy.

## Project Context

This prototype was developed as part of the **EDHEC Open Innovation Challenge**, in collaboration with **Orange Business managers**.

The project focused on the impact of **AI on B2B relationships** through a **Prospect Network Intelligence** approach : understanding stakeholder ecosystems, influence dynamics, and the best commercial entry points within a target account.

## Main Features

- Account portfolio view
- Account mapping interface
- Stakeholder ecosystem analysis
- Identification of key decision-makers, influencers, and blockers
- Deal strategy support
- Business development workflow
- Sales playbook generation
- Admin interface

## Getting Started

### 1. Clone the repository

~~~bash
git clone https://github.com/lbelouan/orange-prospect-navigator.git
cd orange-prospect-navigator
~~~

### 2. Install dependencies

~~~bash
npm install
~~~

### 3. Run the development server

~~~bash
npm run dev
~~~

### 4. Build for production

~~~bash
npm run build
~~~

## Project Goal

The goal of this project is to support **complex B2B sales processes** by making stakeholder ecosystems easier to understand and act upon.

Rather than relying only on static account research, the application helps users :
- visualize influence dynamics
- identify the right stakeholders to approach
- find relevant entry points
- support deal strategy
- structure more targeted commercial action plans

## Application Sections

- **Accounts** : overview of target accounts
- **Account Map** : stakeholder and relationship mapping
- **Strategy** : deal and influence analysis
- **Business Dev** : prospecting-oriented workflow
- **Playbook** : structured action recommendations
- **Admin** : configuration and management interface
